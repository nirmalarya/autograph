# AutoGraph Specifications

This folder contains all specification files for AutoGraph.

## Files

- **app_spec.txt** - Original project specification (654 features)
- **feature_list.json** - Complete feature list with pass/fail status
- **enhancement_spec.txt** - Enhancement/bugfix specifications

## Usage

These files are used by the `autonomous-harness` to guide development and testing.

Do not edit manually unless you know what you're doing!
